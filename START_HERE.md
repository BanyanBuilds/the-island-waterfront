# Start Here

1. Extract this ZIP.
2. Open a terminal in the extracted folder.
3. Run:

```bash
npm install
npm run dev
```

4. Open the local URL Vite prints, normally `http://localhost:5173`.

The project is a React + Vite app. No database or environment variables are required for this first visual build.
